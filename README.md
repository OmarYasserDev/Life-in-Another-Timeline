# Life in Another Timeline

Life in Another Timeline is a creative web application that aims at investigating the difference in living conditions in several countries and cultures.
This website reflects the alternative way of living, learning, and working.

# Features

- Live in:
  - 🇺🇸 The United States
  - 🇯🇵 Japan
  - 🇩🇪 Germany
- Find out about:
  - Education options
  - Working environment
  - Famous places and sights
- User-friendly and visually attractive layout
- Simple and neat user interface

# Built With

- HTML5
- CSS3

# Project Overview

The idea behind this project was to provide an interactive website where a person could understand the possibilities of living in another timeline through several countries and their different lifestyles.
